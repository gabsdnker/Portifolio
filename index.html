Sobre Mim / About Me

Olá! Sou Gabrielli Danker, estudante de Ciência da Computação no Instituto Federal Catarinense, atualmente em formação para me tornar desenvolvedora de software. Tenho interesse em atuar na área de desenvolvimento web e backend, com foco em linguagens como Java e Python, além de experiência em bancos de dados relacionais (MySQL, SQLite, MariaDB). Busco constantemente evoluir minhas habilidades técnicas e práticas, já tendo concluído cursos em programação orientada a objetos, lógica de programação, administração de banco de dados e desenvolvimento de sistemas. Tenho familiaridade com HTML, CSS e JavaScript, além de utilizar ferramentas como Git e GitHub para versionamento de projetos. Gosto de aprender novas tecnologias, enfrentar desafios e colaborar em equipe, sempre buscando entregar soluções que unam qualidade e eficiência.

Assistente de Estudos com Agenda Inteligente

Este projeto é um sistema de gerenciamento de tarefas e projetos voltado para estudantes, que oferece uma interface gráfica simples e sugestões inteligentes baseadas em horários. A aplicação foi desenvolvida em Java e utiliza 6 Padrões de Projeto como parte da arquitetura.

EcoVibe Store

Este projeto é uma loja virtual fictícia voltada para a venda de produtos sustentáveis e ecológicos. Desenvolvido com HTML, CSS e JavaScript, o site conta com sistema de login, carrinho de compras dinâmico, navegação entre páginas e design responsivo. A proposta foi criar uma experiência simples e funcional de e-commerce como forma de praticar conceitos de front-end.


This project is a fictional online store focused on selling sustainable and eco-friendly products. Developed with HTML, CSS, and JavaScript, the website features a login system, dynamic shopping cart, page navigation, and responsive design. The goal was to create a simple and functional e-commerce experience as a way to practice front-end concepts.

BluCidadão

Este projeto é um aplicativo inovador que centraliza os serviços públicos de Blumenau (saúde, educação, emprego, documentos, utilidades e ouvidoria digital com acompanhamento de solicitações). Desenvolvido com Flutter (Dart) no front-end, Java Spring Boot no back-end e MySQL como banco de dados.


This project is an innovative app centralizing Blumenau's public services (health, education, employment, documents, utilities, and digital ombudsman with case tracking). Built with Flutter (Dart) on the front end, Java Spring Boot on the back end, and MySQL as the database.
